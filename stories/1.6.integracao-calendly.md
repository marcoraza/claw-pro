## Story

**As a** lead qualificado para os tiers Pro ou Premium,
**I want** agendar uma call de qualificação de 20 minutos diretamente em uma página sem precisar trocar emails,
**so that** o processo de agendamento seja rápido, conveniente e respeitoso do meu tempo.

1. Link ou embed do Calendly aparece na página de confirmação (`/consultoria/confirmacao`) para leads Pro/Premium
2. Link ou embed do Calendly aparece no E-mail 1 da sequência (Story 1.3 / 1.5)
3. O calendário reflete a disponibilidade real de Marco (configurado diretamente no Calendly)
4. Ao agendar, o cliente recebe confirmação automática por email com link Zoom/Meet (enviado pelo Calendly)
5. Calendly free tier é suficiente para o MVP (sem integração webhook no backend)
6. Link funciona corretamente em mobile (375px)
7. Marco consegue configurar no máximo 3 calls de qualificação por semana via configurações do Calendly (sem código)

## Dev Notes

### Abordagem: Link vs. Embed

Para o MVP (Calendly free tier), usar **link simples** em vez de embed/popup:
- Free tier do Calendly não suporta embed personalizado sem branding
- Link abre em nova aba → experiência aceitável para MVP
- Embed com branding Calendly pode parecer pouco profissional — avaliar com Marco

Se Marco quiser o popup inline, o Calendly tem um script:
```html
<!-- Apenas se free tier suportar -->
<link href="https://assets.calendly.com/assets/external/widget.css" rel="stylesheet">
<script src="https://assets.calendly.com/assets/external/widget.js" async></script>
```
Usar `next/script` com `strategy="lazyOnload"` para não bloquear a página.

### Componente Simples (Opção Recomendada para MVP)

```tsx
// components/consultoria/calendly-embed.tsx
'use client';

interface CalendlyEmbedProps {
  className?: string;
}

export function CalendlyEmbed({ className }: CalendlyEmbedProps) {
  const url = process.env.NEXT_PUBLIC_CALENDLY_URL;
  if (!url) return null;

  return (
    <a
      href={url}
      target="_blank"
      rel="noopener noreferrer"
      className={className}
      aria-label="Agendar call de qualificação de 20 minutos no Calendly"
    >
      Agendar minha call de 20 min →
    </a>
  );
}
```

### Detecção de Tier na Página de Confirmação

A URL de confirmação pode receber um parâmetro para distinguir Essencial de Pro/Premium:
- `/consultoria/confirmacao?tier=essencial` → sem bloco Calendly
- `/consultoria/confirmacao?tier=pro` → com bloco Calendly

Ou: basear-se no registro `consulting_leads` (se disponível via token de sessão).

Opção mais simples para MVP: **parâmetro na URL**, sem consulta ao banco.
