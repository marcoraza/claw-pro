## Story

**As a** cliente que comprou ou agendou a consultoria,
**I want** preencher um formulário detalhado sobre meu negócio antes da sessão,
**so that** Marco chegue completamente preparado e possamos aproveitar ao máximo as 4 horas.

## Acceptance Criteria

1. Formulário com as 8 perguntas do plano da consultoria (ver seção Dev Notes)
2. Acessível via link único gerado por token seguro, enviado ao cliente após pagamento
3. Respostas salvas em `consulting_leads.discovery_form` (campo JSONB)
4. Status do lead atualizado para `'qualified'` após submit bem-sucedido
5. Notificação para Marco (email e/ou Telegram) imediata quando formulário é submetido, com todas as respostas
6. Formulário funciona corretamente em mobile (375px)
7. Token do link único expira após 30 dias ou após o submit (o que vier primeiro)
8. Após submit, exibir mensagem de confirmação: "Pronto! Suas respostas foram enviadas. Marco entrará em contato para confirmar data e hora da sessão."

### As 8 Perguntas do Formulário

Estrutura do campo `discovery_form` (JSONB):
```json
{
  "negocio": "Descrição do negócio e o que vende/oferece",
  "maior_dreno": "Qual é o maior dreno de tempo no dia a dia",
  "tarefas_prioritarias": "Quais tarefas quer que o agente execute primeiro",
  "ferramentas": "Ferramentas já usadas (WhatsApp, Gmail, CRM...)",
  "provider_ia": "Tem conta em OpenAI, Anthropic, Google?",
  "api_key": "Tem chave de API criada, ou prefere criar durante a sessão?",

  "contexto_extra": "Tem algo que preciso saber antes de começar?"
}
```

### Estrutura de Campos na Tabela

Migration a criar:
```sql
ALTER TABLE consulting_leads
  ADD COLUMN IF NOT EXISTS discovery_token TEXT UNIQUE,
  ADD COLUMN IF NOT EXISTS discovery_token_expires_at TIMESTAMPTZ;

CREATE INDEX IF NOT EXISTS idx_consulting_leads_discovery_token
  ON consulting_leads(discovery_token);
```

### Geração do Token (padrão seguro)

```typescript
// Em webhook da Story 1.2, ao confirmar pagamento:
import { randomUUID } from 'crypto';

const token = randomUUID(); // criptograficamente seguro, 128 bits
const expiresAt = new Date(Date.now() + 30 * 24 * 60 * 60 * 1000); // 30 dias

await supabase.from('consulting_leads').update({
  discovery_token: token,
  discovery_token_expires_at: expiresAt,
}).eq('id', leadId);

const discoveryLink = `${process.env.NEXT_PUBLIC_APP_URL}/consultoria/discovery/${token}`;
```
