## O que é o ClawPro

ClawPro é o **guarda-chuva estratégico** de um ecossistema brasileiro completo em torno do OpenClaw. Não é um produto — é uma plataforma com três frentes de negócio integradas que se alimentam mutuamente.

**Tese:** o mercado de wrappers SaaS genéricos de OpenClaw está saturando. A oportunidade está na camada superior: consultoria premium de implementação e um marketplace de skills — ambos com zero concorrência estruturada no Brasil.

---

## As Três Frentes

| Frente | Produto | Ticket | Papel no Ecossistema |

## O Flywheel

```
Conteúdo / Meta Ads / Afiliados
           ↓
  Empresário vê hype do OpenClaw
           ↓
  Consultoria → instalação + treinamento para o negócio
           ↓
  Cliente tem OpenClaw funcionando
           ↓
  Quer mais capacidades → Marketplace de Skills
           ↓
  Skills das consultorias → Marketplace + ClawHub (gratuito)
           ↓
  ClawHub gera descoberta orgânica (320k users)
           ↓  (loop)
```

**TurboClaw** entra no flywheel como alternativa self-service para quem não quer consultoria agora, com upsell natural para as outras frentes ao longo do tempo.

## Contexto de Mercado

**OpenClaw:** framework open-source com 320k stars GitHub, 61k forks. Acesso via Telegram, WhatsApp, Discord e 12+ canais. Suporta Claude, GPT, Gemini, Ollama. Dificuldade técnica real: Docker, Node.js, VPS, SSH — 40–60min para técnico, 13+ dias para não-técnico.

**Concorrência:**
- Wrappers SaaS (TurboClaw, ClawdIA, myQuickClaw): mercado saturando, 3+ players
- Consultoria 1-a-1 estruturada: **ninguém no Brasil**
- Marketplace de skills: **ClawHub vazio**, zero skills de terceiros publicadas
